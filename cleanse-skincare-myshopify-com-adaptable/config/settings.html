Navigation

Manual Linked Lists: Similar to other Shopify themes, this option requires you to create an identically-named Linked List (on the Navigation page) for each Main Menu primary navigation item. Each secondary navigation item would need to be its own collection.

Example: Mens > Footwear > Sample Shoe would need Sample Shoe to be in the Mens and Footwear collections. Mens would be a Main Menu item. Footwear would be within a new Linked List named "Mens".

Generate From Tags: If you add a Collection to the Main Menu primary navigation, this option will generate a secondary navigation from all the Tags that the products within the Collection contain. Works for products/collections only, not pages.

Example: Mens > Footwear > Sample Shoe would need Sample Shoe to be in the Mens collection, and be tagged with "Footwear"
